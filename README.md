Formula One Web Page

A simple web page project dedicated to the world of Formula One. 
This project showcases the key aspects of the sport, including teams, drivers, race tracks, and history. 

This repository contains a basic static website designed to introduce users to Formula One. 
It's ideal for beginners learning web development or anyone interested in presenting structured content about a sport.

The site is organised into sections that hightlight:
Brief Introduction
Drivers
Teams
Tracks
History
